void calculateColorAndFillBuffer ( void  )

Calculates colors for each panel for different color modes.

contains a switch for the color mode:
MODE_OFF: set every color of the whole RingBuffer to 0.
MODE_INDIVIDUAL_COLOR: set the colors of the ringbuffer for every attached module to the received color for this panel
MODE_ONECOLOR: set every color of each attached module to the received colors (every panel has now the same color)
MODE_TWO_COLOR_FADER: calculate "NUM_COLOR_GRADIENTS" of color values (between the received two colors) and save them to the RingBuffer Â 
MODE_TWO_COLOR_GRADIENT: calculate "NUM_NUM_ATTACHED_MOUDLES" of color values (between the received two colors) and save them to the RingBuffer
