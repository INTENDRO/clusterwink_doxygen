Detailed Description

Standard utilities for uC internals (timer,adc,gpio,...)

  • GPIO
    Set pin directions and enable/disable the power LED driver circuit.
  • PWM
    Setup and control the PWM pin used for dimming the power LED
  • ADC
    Measure the voltage of the temperature sensor to determine the power LED temperature.
  • Utilities
    Map function for converting variable to a different number range and a 1ms wait routine.
