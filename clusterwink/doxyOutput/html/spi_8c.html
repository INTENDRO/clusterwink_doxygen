Detailed Description

Initializing the SPI slave interface.

The SPI interface is used to exchange data with the Raspberry Pi (RPi). Only the slave functionality is implemented since the RPi will always be the master. The Interrupt Service Routine (ISR) in the main.c file processes the received data.

void SPISlave_Init ( void  )

Initializes the SPI peripheral as slave.

Enables the SPI peripheral and its interrupt source. Pin directions of !SS, SCK and MOSI are automatically overriden to "INPUT". MISO is user defined and is set as an output to be able to send data back to the master. SCK is idle low and data is sampled on the rising edge of SCK.
